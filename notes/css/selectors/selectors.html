Selectors

Descendant Selectors

Selects all elements which are descendants of another by adding a space between the parent selector and descendant in CSS.

Attribute Selector

Selecting an attribute in a form element such as text.

input[type="text"] { styles here... }

Pseudo Classes

Keywords added to a selector which specifies a special state of the selected element.

    For example:
  • :active
  • :checked
  • :first
  • :first-child
  • :hover
  • :not()
  • :nth-child()
  • :nth-of-type()

Pseudo Elements

Keywords added to a selector which lets you style a particular part of the selected element.

    For example:
  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::selection
